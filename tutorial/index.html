Tutorial membuat github pages melalui git.

1. Membuat repository di github

Buatlah repository di github dengan nama username anda.github.io, misal arethalevi.github.io

2. Membuat repository lokal

Buka terminal (untuk Mac) atau git bash (untuk Windows), pergi ke direktori file yang mau di push ke github menggunakan cd. Disini nama file yang mau saya unggah adalah porto. Lalu, ketik git init untuk membuat repository di file porto

3. Menambahkan isi file lalu commit

Ketikkan git add . untuk memasukkan semua file ke repository, lalu gunakan git commit -m untuk mengcommit file.

4. Mempush file repository lokal ke github

Ketik git remote add origin, lalu diikuti dengan link github yang sudah dibuat di langkah pertama. Lalu untuk mempush gunakan git push -u origin. Nanti akan diminta untuk memasukkan username dan password. Untuk username bisa ketikkan username github, lalu untuk password, isi dengan token. Untuk mendapatkan token dijelaskan di langkah 5

5. Mendapatkan token github

Pada home github, klik gambar profile, pilih Settings, lalu pilih menu Developer Settings, lalu pilih menu Personal access tokens. Setelah itu, klik generate new token (jika anda belum membuat token), centang bagian repo, dan generate token. Nanti akan muncul token, copy saja lalu paste sebagai Password.

6. File berhasil terpush di github

Setelah memasukkan password, file dari porto sudah ada di github.

7. Github pages langsung aktif.

Anda bisa mencoba masukkan di browser anda username.github.io (misal arethalevi.github.io), anda bisa langsung melihat halaman yang sudah anda buat.
Ps: halaman akan lansung muncul bila di dalam file porto, sudah ada file html bernama index.html
